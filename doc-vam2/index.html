文档-Mod+os+ VAM 2.0-2.5

2024-03-09 次阅读 2 min read

在6.1版本中，我们对VAM(变量管理器，variable manager）进行了更新，本文档将对VAM2.0说明。

VAM1.0与VAM2.0有什么更新内容？

在VAM2.0，我们添加了更改变量的功能。

Tips：VAM1.0说明见下面链接文档-mod os+VAM 1.0

使用方法

1.在6.2及以上的系统登录（即已经输入密码，初始化）后，在键盘中按“V”,即可打开“VAM2.0”。

2.在输入框中输入以下内容，会反馈变量内容。

3..在输入框中输入以下内容，会提示输入更改变量的数值。输入后，会更改变量。

字符 变量 备注
Change//zoon zoon
Change//mimastop 密码停止检测
Change//math1 计算1
Change//math1 计算2
Change//user 用户名
Change//push pash
Changey//mid mid码 该变量无法更改。当输入后，会提示无法更改
stop / 关闭VAM 2.0

在编写VAM时，有使用“消息”（“消息”名为“vam”）来启动。（启动过程：按“v”-激活名为“vam”的“消息”-VAM源代码接收到“消息”-启动VAM）

安全提示！！

1.VAM无法反馈与更改“登录密码”与“初始密码”！

2.随意更改变量可能会导致系统故障！！

Tips：详细安全性说明见：安全性-VAM的安全性问题预警

VAM2.5更新

修复了变量“密码停止检测”无法反馈变量内容的问题。
