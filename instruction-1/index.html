使用-mod+os+1.0的使用说明

2024-03-09 次阅读 2 min read

使用说明

1.关机

点击其按钮，将会进入登录模式，需要输入正确的密码

初始密码为:一个空格

2.停止

点击此按钮之后，所有的程序都会停止

3.mp4

这个软件可以听音乐，关于什么歌呢？请自己去猜吧！

↑提升音量

↓减少音量

4.天气

你可以输入你所在的城市，获取天气

5.猜一猜

系统会内定一个1至100的数字，你可以去猜一猜这个数字是什么哦？一般来说是猜不到的。嘿嘿(º﹃º )

6.翻译

你可以输入中文，系统将会翻译成英文

7.时间

将会写出您当前时间，按北京时间计算

8.查找序列号

你只需要按下空格键，会显示版本序列号以及作者大大的名字（kegongteng是原作者）

9.可能新加的内容

暂无

10.部分软件需要用的条件

1.天气以及翻译需要网络以便连接服务器。

2.建议使用mid+这个软件，防止出bug。（下载：http://mindplus.cc/ )

10.r键

这是一个像windows的运行。

11.开源

大家可以尽情的改编这个软件，最好把你名称写上去哦

变量

1.用户名：给使用者看的用户名变量

2.mozoo：密码变量

3.pash：猜一猜的变量

4.zoon：给系统的用户名变量

5.my variable ：无用
